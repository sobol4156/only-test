## Описание

Реализован блок отображения временных отрезков с ключевыми событиями в каждом отрезке.  
Особенности блока:

- В блоке от 2 до 6 временных отрезков.
- При переключении временных отрезков меняются числа, а под ними отображается слайдер с подробной информацией по событиям.
- Все интерактивные точки на окружности равномерно расположены.
- Блок независим от остальной логики страницы. Можно добавлять несколько блоков на страницу, и они будут работать корректно.
- Все линии макета реализованы как часть верстки, а не вспомогательная сетка.
- Демонстрация работы блока осуществляется с максимальным количеством временных отрезков.

---

## Стек технологий

- **React.js** с TypeScript
- **Swiper** — слайдер для отображения событий
- **styled-components** — стилизация компонентов
- **Webpack** — сборка проекта

---

## Особенности реализации

- Компоненты блока полностью независимы, их можно добавлять несколько на страницу.
- Слайдеры реализованы на базе Swiper с кастомной навигацией и точками.
- SVG и шрифты подключены через Webpack (копируются в билд через `copy-webpack-plugin`).

## Установка

1. Клонировать репозиторий:

```bash
git clone https://github.com/sobol4156/timeline-block.git
```

2. Перейти в папку проекта:

```bash
cd only-test
```

3. Установить зависимости:

```bash
npm install
```

## Запуск проекта
Локальная разработка:

```bash
npm run start
```
